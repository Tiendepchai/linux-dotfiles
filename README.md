# linux-dotfiles

Frozen snapshot of the current Linux user configuration from `/home/nuke`.

Included in this snapshot:

- Shell configs: `.zshrc`, `.bashrc`, `.bash_profile`, `.gitconfig`
- Prompt and terminal configs: `starship`, `kitty`, `fuzzel`
- Desktop and compositor configs: `waybar`, `mako`, `niri`, `swaylock`
- Editor config: `nvim`
- Desktop preferences: `mimeapps.list`, `user-dirs.dirs`, `pavucontrol.ini`

Intentionally omitted:

- Shell history and other personal runtime state
- Large application state directories under `.config`
- Nested Git metadata from local config folders
- Wallpaper assets referenced by config files
